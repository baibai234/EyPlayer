## 项目结构

```
EyPlayer/
├── src/
│   ├── main/                     # Electron 主进程
│   │   └── index.ts              # 主进程入口 (窗口管理, IPC, 播放器调用)
│   ├── preload/
│   │   └── index.ts              # 预加载脚本 (API暴露)
│   ├── renderer/
│   │   ├── components/           # 公共组件
│   │   │   ├── common/           # Loading, Error
│   │   │   ├── layout/           # Navbar
│   │   │   └── media/            # MediaCard, MediaList, Carousel
│   │   ├── views/                # 页面视图
│   │   │   ├── Home.vue          # 主页 (轮播, 分类列表)
│   │   │   ├── Media.vue         # 媒体库 (筛选, 网格展示)
│   │   │   ├── Watching.vue      # 在看 (进度条, 继续播放)
│   │   │   ├── Search.vue        # 搜索 (最近搜索, 热门搜索)
│   │   │   ├── Settings.vue      # 设置 (服务器, 播放器, 外观)
│   │   │   └── Detail.vue        # 详情 (海报, 演员, 播放)
│   │   ├── composables/          # 组合式函数 (useEmby, usePlayer, useCache)
│   │   ├── stores/               # 状态管理 (emby, player, settings)
│   │   ├── services/             # 服务层 (emby-api, player-service, cache-service, strm-parser)
│   │   ├── utils/                # 工具函数 (file, network, format)
│   │   └── types/                # 类型定义 (emby, player, electron)
│   └── shared/                   # 共享代码 (constants, types)
├── devlog/                       # 开发日志
├── docs/                         # 项目文档
└── CLAUDE.md                     # Claude Code工作指引
```

---

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **桌面**: Electron 28.x
- **状态管理**: Pinia
- **路由**: Vue Router
- **代码规范**: ESLint + Prettier

---

## 核心功能

1. **Emby服务器连接** - API封装, IPC代理, 跨域请求
2. **媒体库展示** - 轮播, 分类列表, 网格布局, 横向滚动
3. **播放功能** - STRM解析, MPV/PotPlayer/VLC调用, 进度同步
4. **媒体管理** - 搜索, 收藏, 在看, 标记已看/未看
5. **界面设计** - 毛玻璃效果, 苹果风格, 响应式布局
6. **设置配置** - 服务器, 播放器, 外观, 缓存管理
