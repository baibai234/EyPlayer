# EyPlayer 代码规范文档

## 一、通用规范

### 1.1 文件编码
- 所有文件使用 UTF-8 编码
- 换行符使用 LF (Unix 风格)

### 1.2 缩进规范
- 使用 2 个空格缩进
- 不使用 Tab 字符
- 保持一致的缩进风格

### 1.3 行宽限制
- 每行代码不超过 100 个字符
- 超长行进行合理换行
- 保持代码可读性

### 1.4 空行规范
- 函数之间空一行
- 逻辑块之间空一行
- 文件末尾保留一个空行

### 1.5 空格规范
- 运算符两侧加空格
- 逗号后面加空格
- 冒号后面加空格（对象字面量）
- 分号后面加空格
